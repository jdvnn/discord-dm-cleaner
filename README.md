# Discord DM Cleaner

**A lightweight utility for cleaning up your own Discord Direct Messages.**

Fast • Simple • Open Source

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Node.js](https://img.shields.io/badge/node-%3E%3D18-green)
![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20Linux%20%7C%20macOS-lightgrey)

## ✨ Features

- 🗑️ Delete your own Discord DM messages
- ⚡ Fast and lightweight
- 🔒 Only targets messages sent by your account
- 📊 Live progress output
- 🛡️ Built-in rate-limit handling
- 💻 Cross-platform support

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/jdvnn/discord-dm-cleaner.git
cd discord-dm-cleaner
````

Install dependencies:

```bash
npm install
```

---

## 🚀 Usage

Start the application:

```bash
node selfbot.js
```

Follow the on-screen instructions to configure the cleaner.
